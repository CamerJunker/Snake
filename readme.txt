# English
# Run the program
Open the "Snake"-folder and run this command:

Windows:
```
java -jar .\target\snake.jar n m
```

Linux / MacOS:
```
java -jar target/snake.jar n m
```

Where n and m are integers from 5 to 100 each, not literally "n m".


# Dansk
## Kør programmet
Navigér ind i Snake-mappen og kør med denne kommando:

Windows:
```
java -jar .\target\snake.jar n m
```

Linux / MacOS:
```
java -jar target/snake.jar n m
```

Hvor n og m er heltal fra 5 til 100 hver, ikke bogstaveligtalt "n m".
